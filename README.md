# Sistema Bancário em Java

Projeto desenvolvido com o objetivo de praticar os fundamentos da linguagem Java e Programação Orientada a Objetos.

O sistema simula operações básicas de uma agência bancária, permitindo a criação e gerenciamento de contas diretamente pelo terminal.

## Funcionalidades

- Criação de contas bancárias
- Depósito
- Saque
- Transferência entre contas
- Listagem de contas cadastradas

## Conceitos aplicados

- Classes e objetos
- Encapsulamento (private, getters e setters)
- Construtores
- Métodos estáticos
- Listas com ArrayList
- Fluxo de controle e validações
- Organização de um projeto Java simples

## Tecnologias

- Java
- IDE: VS Code
- Execução via terminal
